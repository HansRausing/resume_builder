# Resume Templates Folder

## 📄 How to Use

1. **Export your resume as PDF** from Word, Google Docs, or your favorite tool
2. **Rename it to:** `resume-template.pdf`
3. **Place it here:** `frontend/public/templates/resume-template.pdf`
4. **Done!** The app will automatically use your template

## 📋 Template Requirements

Your template should have:
- Clear section headers (PROFESSIONAL SUMMARY, WORK EXPERIENCE, EDUCATION, SKILLS)
- Professional formatting
- Standard fonts (Arial, Calibri, Times New Roman, Helvetica)
- Text-based content (minimal graphics)
- One or two pages

## 🎯 What Happens

When you download a PDF:
1. App loads your template
2. Replaces Summary, Experience, and Skills sections
3. Keeps your exact formatting
4. Downloads customized PDF

## 💡 No Template?

No problem! If you don't add a template:
- App generates a professional PDF from scratch
- Clean, ATS-friendly formatting
- Still looks great!

## 🔧 Troubleshooting

**PDF not loading?**
- Check filename is exactly: `resume-template.pdf`
- Ensure it's a valid PDF file
- Try simplifying the template (remove complex layouts)

**Formatting looks off?**
- Use simpler template layout
- Avoid tables and multi-column layouts
- Use standard fonts
- Ensure clear section headers
